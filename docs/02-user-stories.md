# QuizTech - User Stories (50 Stories)

## Student (Sinh viên) - 25 stories

### Auth
1. As a student, I want to register an account so that I can use the system
2. As a student, I want to login with email or student code so that I can access my account
3. As a student, I want to logout so that I can secure my account
4. As a student, I want to reset my password so that I can recover my account
5. As a student, I want to view my profile so that I can see my information

### Exam
6. As a student, I want to view all available exams so that I can choose which to take
7. As a student, I want to filter exams by subject so that I can find relevant exams
8. As a student, I want to start an exam so that I can test my knowledge
9. As a student, I want to see a timer during the exam so that I know how much time is left
10. As a student, I want to flag a question for review so that I can come back to it later
11. As a student, I want to submit my exam so that I can get my score
12. As a student, I want to view my exam results so that I can see my performance
13. As a student, I want to review correct answers so that I can learn from my mistakes
14. As a student, I want to see AI explanation for answers so that I understand the reasoning

### Dashboard
15. As a student, I want to see my dashboard so that I can track my progress
16. As a student, I want to see my exam history so that I can review past performances
17. As a student, I want to see my statistics (total exams, avg score) so that I know my level
18. As a student, I want to see recommended exams so that I know what to take next

### Room (Competition)
19. As a student, I want to join a room with a code so that I can compete with others
20. As a student, I want to see other players in the room so that I know who I'm competing with
21. As a student, I want to answer questions in real-time so that I can compete
22. As a student, I want to see the leaderboard updating live so that I know my rank

### Learning
23. As a student, I want to see my weak subjects so that I know what to study
24. As a student, I want to get AI recommendations for learning so that I can improve
25. As a student, I want to review flagged questions so that I can focus on difficult topics

## Teacher (Giảng viên) - 15 stories

### Subjects
26. As a teacher, I want to create subjects so that I can organize questions
27. As a teacher, I want to edit subjects so that I can update information
28. As a teacher, I want to delete subjects so that I can remove unused ones

### Questions
29. As a teacher, I want to create questions so that I can build question bank
30. As a teacher, I want to edit questions so that I can fix errors
31. As a teacher, I want to delete questions so that I can remove outdated ones
32. As a teacher, I want to import questions from image using OCR so that I can save time
33. As a teacher, I want to generate questions with AI so that I can create exams quickly
34. As a teacher, I want to categorize questions by difficulty so that I can create balanced exams

### Exams
35. As a teacher, I want to create exams so that students can take tests
36. As a teacher, I want to add questions to exams so that I can build test content
37. As a teacher, I want to set exam duration so that I can control time
38. As a teacher, I want to activate/deactivate exams so that I can manage availability
39. As a teacher, I want to view student results so that I can assess performance
40. As a teacher, I want to export results so that I can use for grading

### Rooms
41. As a teacher, I want to create rooms so that students can compete
42. As a teacher, I want to manage rooms so that I can control the competition
43. As a teacher, I want to start/end rooms so that I can manage sessions

## Admin - 7 stories

44. As an admin, I want to view all users so that I can manage accounts
45. As an admin, I want to block/unblock users so that I can enforce rules
46. As an admin, I want to assign roles so that I can manage permissions
47. As an admin, I want to view system statistics so that I can monitor usage
48. As an admin, I want to manage AI logs so that I can monitor AI usage
49. As an admin, I want to configure system settings so that I can customize the platform
50. As an admin, I want to backup/restore database so that I can prevent data loss
