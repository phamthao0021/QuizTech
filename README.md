# QuizTech - Hệ thống thi trắc nghiệm CNTT

## Giới thiệu

QuizTech là hệ thống thi trắc nghiệm trực tuyến dành cho sinh viên Công nghệ thông tin, với các tính năng:

- 📝 Thi trắc nghiệm trực tuyến
- 🏆 Thi đấu theo phòng (Realtime)
- 🤖 AI hỗ trợ sinh câu hỏi và phân tích
- 📊 Dashboard cá nhân và bảng xếp hạng
- 👨‍🏫 Quản lý ngân hàng câu hỏi và đề thi

## Công nghệ sử dụng

- PHP 8.x
- MySQL 8.x
- HTML5, CSS3, Bootstrap 5
- JavaScript, jQuery
- RESTful API
